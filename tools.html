Tools zur OER-Erstellung im Überblick

Kompakter Überblick über einige Best-Of Tools (überwiegend Open Source), die sich zur Erstellung von OER gut eignen und auch von Einsteigerinnen einfach nutzbar sind.

Geplanter Input beim Coaching-Termin

Folgende Tools werden insbesondere vorgestellt. Vertiefungen und Erweiterungen je nach den Interessen/ Bedarfen der Teilnehmenden:

  • H5P ist eine sehr vielfältig nutzbare Open Source Software mit dem Fokus der Erstellung interaktiver Online-Lernmaterialien. Das Tool umfasst mehrere so genannte Inhaltstypen. Dazu zählen unter anderem interaktive Videos, Quizze, Karteikarten oder Drag and Drop Aufgaben. Damit sind die Möglichkeiten von H5P aber noch nicht erschöpft. Das Tool lässt sich z.B. auch für Projektplanung oder für Präsentationen nutzen. H5P Materialien werden im Format .h5p erstellt. Dieses Format kann online genutzt, beliebig auf andere Websites eingebettet und mithilfe eines Editors erstellt und remixt werden. Der Editor ist als Plugin für Drupal, Moodle, Ilias und Wordpress verfügbar. Auf der Website EinstiegH5P.de kannst Du erste H5P-Inhalte ohne Registrierung testweise erstellen. Für eine dauerhafte Speicherung muss ein kostenfreier Account angefragt werden. Zum Starten eignen sich simple Inhaltstypen am besten - z.B. Multiple Choice.
  • Mit HackMD lassen sich Webseiten erstellen, teilen, kollaborativ bearbeiten und auch über einen Slide-Modus präsentieren. Einige Funktionalitäten stehen erst nach kostenfreier Registrierung zur Verfügung. HackMD basiert auf der Open Source Software CodiMD. Bei fortgeschrittenen technischen Kenntnissen kann diese auch selbst installiert werden. Ich empfehle die Nutzung der Installation der Open Knowledge Foundation
  • Glitch präsentiert Code in Form von direkt nutzbaren Webanwendungen. Da erstellte Projekte leicht remixt und neu veröffentlicht werden können, ist Glitch auch für technisch weniger-erfahrene Menschen spannend. Ausführlich erklärt ist Glitch in einem Blogbeitrag im eBildungslabor. Es lohnt sich, dem Twitter-Account von Glitch für Updates zu folgen.
  • Etherpads basieren auf der gleichnamigen Open Source Software, die die Erstellung von beschreibbaren Webseiten im Internet ermöglicht. Diese können dann über einen Link geteilt und gemeinsam bearbeitet werden. Das entstehende Dokument ist für alle online abrufbar, die über den Link verfügen. Mit Etherpads lassen sich vor diesem Hintergrund gemeinsam OER erstellen und direkt teilen - etwa via ZumPad. Eine Alternative zum klassischen Etherpad sind GoogleDocs. Diese ermöglichen übersichtliche Kommentierungen und Bearbeitungen. Ähnliche Funktionalitäten bietet im Open Source Bereich Collabora Online und Only Office (z.B. im Rahmen der Nextcloud). Eine Etherpad-Alternative mit Fokus auf Datenschutz ist das Cryptpad. Eine Erweiterung (u.a. um Bild-Upload) ist Board.net Hilfreich für eine Veröffentlichung als Website ist WriteURL.com
  • Beim Kreislauftool handelt es sich um eine einfache Web-Anwendung, mit der prozesshafte Kreisläufe aufgezeichnet, animiert und geteilt werden können. Das Teilen erfolgt dabei über einen Link, in dem die Inhalte des jeweils gestalteten Kreislaufs gespeichert sind. Mit dem Tool, können z.B. OER zum Bereich Bildung für Nachhaltige Entwicklung gestaltet werden. Es eignet sich besonders gut zur eigenen Erstellung von OER durch Lernende. Nähere Erläuterungen findest Du in einem Blogbeitrag im eBildungslabor.
  • Mit Tutory lassen sich mittels eines einfachen Drag and Drop Editors ansprechende Arbeitsblätter erstellen und teilen. Über Schnittstellen lassen sich offen lizenzierte Bilder aus dem Internet einfügen. Tutory ist nicht nur ein Tool, sondern auch ein Bildungs-Startup, das als solches mögliche Geschäftsmodelle für OER erkundet. Der Ansatz ist es, das Teilen von Arbeitsblättern zu unterstützen, indem hauptsächlich private Arbeitsblätter kostenpflichtig sind. Das Tool setzt aber in jedem Fall eine Registrierung voraus. In der Basis-Version ist die Nutzung kostenfrei.
  • Twine ist ein Open Source Tool zur Erstellung interaktiver Texte. Vor diesem Hintergrund kann es genutzt werden für Geschichten mit mehreren Handlungssträngen, für Tutorials oder für Spiele. Auf der Website Twinery werden erstellte Twines nur lokal gespeichert. Zur Weiternutzung können sie als HTML-Datei exportiert und als solche beliebig im Internet veröffentlicht werden. Eine explizites Angebot hierfür ist Philome.la. Gut geeignet zum Einstieg ist auch Mein TwineMehr über Twine kannst Du in diesem mit Twine erstellten Tutorial über Twine lernen.
  • Bei RevealJS handelt es sich um ein Open Source Präsentationstool. Gestaltet werden die Präsentationen im HTML-Format. Auf diese Weise können sie einfach weiter genutzt werden. Es gibt eine nette Features wie z.B. einen Gif-Hintergrund und unterschiedliche Animationseffekte. Empfehlenswert ist eine Veröffentlichung über Github Pages. Über diesen Weg können die Präsentationen sowohl als HTML geteilt, als auch direkt online präsentiert werden. Wenn dieser Weg einmal bekannt ist, dann eignet er sich auch darüber hinaus für die Veröffentlichung von OER in unterschiedlichen Formaten. Die Nutzung von RevealJS ist im eBildungslabor-Blog ausführlich beschrieben.
